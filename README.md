# Microsoft Defender for Endpoint - Laboratório

Este diretório contém estudos de caso e playbooks relacionados ao **Microsoft Defender for Endpoint**, simulando cenários de detecção e resposta em ambiente controlado.

## Objetivo
Demonstrar práticas de:
- Detecção de ameaças
- Análise de alertas
- Contenção e remediação
- Aplicação de políticas (ASR, Tamper Protection)
